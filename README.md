# mention-CGU-ADULLAM

Ce dépôt contient les mentions légales et les Conditions Générales d'Utilisation (CGU) de la plateforme Adullam (adullam.dev).

## Contenu
- **index.html** : Page web des CGU Adullam, design moderne et responsive, avec informations légales, société, contact, etc.
- **logo.jpeg** : Logo officiel utilisé dans la page.

## Aperçu
La page CGU est conçue pour être claire, professionnelle et conforme aux exigences légales, avec :
- Présentation de l'entreprise (OFCEAB STUDIO, SARL, RCCM, etc.)
- Table des matières interactive
- Articles détaillés (définitions, obligations, données, tarification, etc.)
- Section contact avec icônes modernes

## Utilisation
1. Ouvrez `index.html` dans votre navigateur pour visualiser la page CGU.
2. Modifiez le contenu selon vos besoins spécifiques (informations société, contacts, etc.).

## Déploiement
Vous pouvez héberger ce fichier sur GitHub Pages ou tout autre service d'hébergement statique.
